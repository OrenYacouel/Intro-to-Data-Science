# Intro-to-Data-Science
**Syllabus**

This course aims to provide a general introduction to the field of data science, and will review some basic tools and ideas that are needed in this field. Thecourse will have a short introduction to scientific programing, a short rehearsal on linear algebra, algebraic least squares minimization, linear regression from a statistical point of view, machine learning principles, examples of classification of objects (supervised), clustering of data and an example of its use for unsupervised learning, introduction to optimization methods.
